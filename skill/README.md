# BNBot Skill

Claude Code skill for BNBot — provides AI agents with X/Twitter growth workflows.

## Usage

```bash
# In Claude Code
/bnbot help
/bnbot "write a tweet about AI agents"
/bnbot "scrape trending topics from Reddit and Twitter"
```

## Setup

```bash
npx @bnbot/cli setup
```

This installs the skill into your Claude Code environment and configures the CLI tools.

## What It Does

- Tweet creation with AI (single tweets, threads, articles)
- Cross-platform content reposting
- Trend scraping from 30+ platforms
- Draft management and scheduling
- Image generation for tweets
