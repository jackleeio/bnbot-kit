# BNBot Web

Official website for BNBot at [bnbot.ai](https://bnbot.ai). Built with Next.js.

## Development

```bash
npm install
npm run dev     # Dev server (port 3000)
npm run build   # Production build
```

## Pages

- `/` — Landing page with features, pricing, quick start
- `/s/[shareKey]` — Public schedule calendar view
- `/s/d/[shareKey]` — Tweet draft preview (Twitter-style card)
- `/docs` — Documentation
