# 最终提示词（KOL内容文章）

写一篇可发布的加密行业分析文章。

要求：
1. 标题根据正文内容自然生成，不能套模板。
2. 先输出“10条去重总结”，再输出“扩展长文正文”。
3. 文章必须是长文，信息密度高，不要短稿。
4. 内容必须围绕推文里真实出现的关键词（项目名、代币名、产品名、事件名）展开。
5. 不要出现“分析数据/样本/抓取结果/去重过程”等后台过程描述。
6. 不要写“主题一/主题二”或 Day1/Day7 计划型结构。
7. 引用自然融入正文，直接附推文链接，不使用“引用：”字样。
8. 语气采用资深 Crypto KOL 风格：有判断、有观点、有市场感。
9. 避免重复表达，避免空话，确保可读性与可发布性。
10. 不要加入任何广告、导流文案或推广链接。
11. 仅输出一篇正常文章格式，不要出现“English/中文”分段标题。
